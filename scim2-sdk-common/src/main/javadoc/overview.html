This package contains shared classes that may be used by both SCIM 2 clients and SCIM 2 service providers. These include:

  • The {@link BaseScimResource} class, from which model classes representing SCIM resource types can be derived.
  • The {@link GenericScimResource} class, which provides a general-purpose interface for accessing attributes of any SCIM resource.
  • The {@link Path} class, which is used to specify the targets of SCIM PATCH operations or to specify attributes to retrieve or modify when manipulating resources through the {@link GenericScimResource} class.
  • The {@link Filter} class and related classes, used to build SCIM 2 query filters.
  • Classes representing SCIM 2 messages, including error responses, search responses, and PATCH operations.
  • Exception classes representing HTTP responses from SCIM 2 service providers.
